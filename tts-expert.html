Privacy Policy for TTS Expert

Application Package: com.neurontslabs.ttsexpert • Effective Date: 12 July 2026

1. Privacy-First Philosophy Statement

Welcome to TTS Expert. At Neuron TS Labs, we engineer our software solutions using strict privacy-by-design protocols. TTS Expert is structurally built to execute high-fidelity audio synthesis directly inside your device sandbox environment. By removing dependencies on permanent remote servers for data processing, we ensure that your written files and generated compositions remain strictly within your autonomous control.

2. Complete Data Non-Collection Manifest

TTS Expert operates transparently without account registries, token registration protocols, or user logins. We do not intentionally harvest, inspect, or retain any personal identifiers or device variables. The following arrays are completely isolated from our software scope:

3. Localized Synthesis Pipeline & Audio Ownership

Text structures typed, pasted, or loaded into the TTS Expert UI engine follow strict handling rules:

  • Processing Domain: Text strings are consumed strictly inside local processing components to compute real-time speech synthesis. No text data parsed by this utility is ever transmitted to, cached by, or sold by Neuron TS Labs.
  • Audio Export Nomenclature: Synthesized outputs transformed into persistent file formats (such as .wav) are stored directly on your physical hardware container using immutable, timestamp-delimited filenames to prevent unintended data loss:
    TTS_20260712_103025.wav
    TTS_20260712_154512.wav
  • Asset Sovereignty: You retain exclusive, unencumbered ownership of all generated audio files, speech assets, and wave outputs. We exercise no software rights over your audio exports.

4. Text-to-Speech Subsystem Framework Architecture

To process synthesis calls effectively while maintaining strict platform boundaries, TTS Expert routes events across established system containers:

Subsystem Layer Operational Scope Privacy Governance Boundary
Android Text-to-Speech Framework Coordinates low-level platform APIs to map string buffers into audible waveforms. Governed natively by the core Android system configuration parameters. Data operations occur strictly inside local memory structures.
Google Text-to-Speech Engine Optional native high-fidelity module used to compute complex natural voice variations. Operates under Google's independent privacy guidelines. If an external online voice configuration is chosen, data handling shifts entirely to that provider.
Custom User Modules Third-party synthesis plug-ins or system voice packs manually installed by the user. Subject exclusively to the localized configuration profiles and terms of those specific module creators.

5. Structural Permission Isolation

TTS Expert operates cleanly without requesting dangerous permissions. The application package explicitly runs without hooks into high-risk system subsystems, including the Device Camera, Microphone, GPS Location sensors, and Contact aggregators. The app calls upon local filesystem operations exclusively to import baseline files or write out high-fidelity audio assets into your designated target directory.

6. Monetization & Analytic Framework Disclosures

To support independent utility development, TTS Expert integrates standard mobile monetization components via the Google AdMob SDK framework. These tools manage ad frequencies and verify interaction integrity independently from our core audio codebase. Data variables captured directly by the advertising layer include:

  • Device model descriptors, active configurations, and hardware parameters.
  • Standard Mobile Advertising IDs (AAID) for frequency monitoring.
  • Basic app interaction analytics and crash diagnostics metrics.
  • Coarse, network-level approximate location strings based on active IP groups.

7. Data Sovereignty, Control, and Erasure Rights

Because your synthesis strings and audio assets are never recorded on external servers, Neuron TS Labs cannot execute remote deletion commands. You retain complete, autonomous authority over your data lifecycles. Wiping application data, managing files, clearing storage caches, or performing a package uninstallation completely deletes all localized system parameters instantly.
